# Status

The current status of the fine-tuning job, which can be either `validating_files`, `queued`, `running`, `succeeded`, `failed`, or `cancelled`.


## Values

| Name               | Value              |
| ------------------ | ------------------ |
| `VALIDATING_FILES` | validating_files   |
| `QUEUED`           | queued             |
| `RUNNING`          | running            |
| `SUCCEEDED`        | succeeded          |
| `FAILED`           | failed             |
| `CANCELLED`        | cancelled          |
