# CreateSpeechRequestResponseFormat

The format to audio in. Supported formats are `mp3`, `opus`, `aac`, `flac`, `wav`, and `pcm`.

| ------ | ------ |
| `MP3`  | mp3    |
| `OPUS` | opus   |
| `AAC`  | aac    |
| `FLAC` | flac   |
| `WAV`  | wav    |
| `PCM`  | pcm    |
