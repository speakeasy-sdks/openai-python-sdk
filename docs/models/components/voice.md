# Voice

The voice to use when generating the audio. Supported voices are `alloy`, `echo`, `fable`, `onyx`, `nova`, and `shimmer`.


## Values

| Name      | Value     |
| --------- | --------- |
| `ALLOY`   | alloy     |
| `ECHO`    | echo      |
| `FABLE`   | fable     |
| `ONYX`    | onyx      |
| `NOVA`    | nova      |
| `SHIMMER` | shimmer   |
