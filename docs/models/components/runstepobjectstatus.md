# RunStepObjectStatus

The status of the run step, which can be either `in_progress`, `cancelled`, `failed`, `completed`, or `expired`.


## Values

| Name          | Value         |
| ------------- | ------------- |
| `IN_PROGRESS` | in_progress   |
| `CANCELLED`   | cancelled     |
| `FAILED`      | failed        |
| `COMPLETED`   | completed     |
| `EXPIRED`     | expired       |
